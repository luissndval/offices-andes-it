# Glosario

Términos compartidos por todos los agentes. Si aparece una sigla nueva, agregarla acá.

## Scrum

- **PO**: Product Owner
- **AF**: Analista Funcional
- **TL**: Tech Lead
- **DoR**: Definition of Ready
- **DoD**: Definition of Done
- **PBI**: Product Backlog Item
- **US**: User Story
- **CA**: Criterios de Aceptación
- **SP**: Story Points
- **MVP**: Minimum Viable Product

## Técnico

- **ADR**: Architecture Decision Record
- **PR**: Pull Request
- **CI/CD**: Continuous Integration / Continuous Delivery
- **IaC**: Infrastructure as Code
- **SAST**: Static Application Security Testing
- **DAST**: Dynamic Application Security Testing
- **SLO**: Service Level Objective
- **SLI**: Service Level Indicator

## Mobile

- **RN**: React Native
- **API min**: API mínima de Android soportada
- **iOS min**: versión mínima de iOS soportada
- **OTA**: Over-the-Air update (típico en RN/Expo)

## Azure DevOps

- **AB#1234**: forma de referenciar un work item en commits y PRs.
- **PBI**: en Azure Boards, equivalente a User Story en proceso Scrum.
