# NOTICE

This project (Agent Workbench) adopts code, assets and patterns from
**pixel-agents** (https://github.com/pablodelucca/pixel-agents) by Pablo De Lucca,
released under the MIT License. Full license text available at
`LICENSES/pixel-agents-MIT.txt`.

Reused components (planned, progressively adopted from Phase 1):

- Office renderer (Canvas 2D engine, sprites, game loop) — `frontend/office/`
- Visual assets (characters, furniture, tiles, manifests) — `frontend/public/assets/`
- UI components (toolbar, modals, zoom controls) — `frontend/components/ui/`

Character sprites are based on the *Metro City free topdown character pack* by
JIK-A-4; see upstream attribution in the original pixel-agents repo.

See `docs/technical/adr/ADR-003-estrategia-de-fork.md` for the rationale of the
copy-with-attribution strategy vs a full fork.
